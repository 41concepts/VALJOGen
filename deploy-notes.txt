Build procect:
mvn package

Build website
cd site
./build_site.sh

Sign:
mvn mvn verify -Dgpg.passphrase=thephrase

Change version (example)
mvn versions:set -DnewVersion=1.0.0-RC2

Deply:
mvn deploy -Dgpg.passphrase=x/y

SETUP:

Configure SonaType account in ~/.m2/settings.xml:

<?xml version="1.0" encoding="UTF-8"?>
<settings>
  <servers>
    <server> <!-- SonaType -->
      <id>ossrh</id>
      <username>myUserName</username>
      <password>myPASSWORD</password>
    </server>
  </servers>
</settings>
